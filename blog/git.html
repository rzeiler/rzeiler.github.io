Ubuntu und Git und Gui

Installieren von Git und Git Gui

In kurzen Schritten erklärt, wie man git und git gui unter Ubnutu Linux installiert.

Installieren

  1. GIT
sudo apt-get install git
1
  1. GIT GUI
sudo apt-get install git-gui
1
  1. Sprachunterstützung (Optional für Deutsch)

Wenn die [git gui] mit der deutschen systemsprache in berührung kommt, dann such diese nach einer Rechtschreibhilfe. In diesem Fall benötigen wir aspell-'langauge'.

sudo apt-get install aspell-de
1
